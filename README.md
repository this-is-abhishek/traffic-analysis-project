# Traffic Analysis Project

## About

This project analyzes traffic data and predicts traffic volume using Python.

## What I used

* pandas, NumPy
* matplotlib
* scikit-learn

## Model

Used Average Speed and Congestion Level to predict Traffic Volume.

R² Score: ~0.78

## How to run

```
pip install -r requirements.txt
python main.py
```

## Output

* Cleaned data in `data/processed/`
* Graphs in `outputs/plots/`
